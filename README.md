# MALTO Hackathon: AI vs Human Text Classification

## Overview

In this hackathon, participants are challenged to build machine learning models that can identify whether a piece of text is written by a human or generated by an AI system. If the text is AI-generated, participants must also predict which specific model generated it.

## The Task
Given a text sample, the model must classify it into one of the following categories:
* Human-written text
* ChatGPT
* Gemini
* Claude
* Grok
* DeepSeek

This is a **multi-class classification task**, where the goal is to correctly identify the author type or the specific generating model. 

### Evaluation Metric
Submissions will be evaluated using the **Macro F1 Score**.
